# Supply Chain Visibility & Optimization – Milestone 2

## Project Overview
This project contains two Power BI dashboards:
- Inventory Analytics Dashboard
- Delivery Performance Dashboard

The dashboards help analyze inventory performance and delivery efficiency using DAX measures, KPIs, and interactive visualizations.

## Inventory Analytics
- Inventory Turnover is calculated using sales and inventory values.
- Slow-moving products are identified based on low sales and high stock.
- Fast-moving products are identified based on high sales and frequent demand.
- The dashboard monitors inventory value, stock quantity, safety stock, reorder level, warehouse, and category performance.

## Delivery Performance
- The dashboard analyzes On-Time and Delayed Deliveries.
- Measures Delivery Lead Time.
- Evaluates Late Delivery Risk.
- Tracks delivery performance by region and shipping mode.
- Shows delivery trends over time with drill-down analysis.

## Key Insights
- Improved inventory monitoring.
- Identified slow-moving and fast-moving products.
- Tracked delivery performance across regions.
- Monitored late deliveries and delivery risks.

## Tools Used
- Power BI
- DAX
- Power Query
